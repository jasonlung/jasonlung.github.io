Chrome extension - Reader View

· 01:54am Oct 06 2022 · # chrome extension

最近在找 Chrome 的 reader mode extension, 安裝並試用了幾個評價較多也較高分的，最後留下
Reader View，甚至覺得它比其他 extensions 好太多了。

特點

  1. 雖然內建不少可以調整的參數，要手動修改 CSS 也沒問題，但預設模式已經很適合閱讀，不輸 Evernote Web Clipper 當閱讀模式使用的效果。
  2. 內建簡易的文書編輯器和便利貼，進入閱讀模式後，可以直接修改文章或劃記重點，並在文章各個位置貼上便利貼。跳出閱讀模式再重新進入，修改過的內容會消失、恢復原文內容，但即使重啟 Chrome 再進入閱讀模式，便利貼還是會在。
  3. 可以把閱讀模式中的內容和格式儲存成 HTML 和 PDF, 包括便利貼，前者的話，便利貼也就是一段語法，後者的話，如果便利貼在閱讀模式中是貼在文字上，存成 PDF 時會蓋掉文字，如果放文章兩側的 margin 再把 PDF 存成 A4 或 US Letter 大小，則很可能不會把便利貼存進去，但假如把 PDF 存成 A2 大小，文字在整個畫面中的比例會變小，兩旁的 margin 就出來了，貼在這的便利貼也就能顯現，特別像是讀 paper 做筆記。

後記

Reader View 是免費的，不要對筆記或劃重點之類的附加功能有太高的期待，如果純粹使用閱讀模式，並把文章存成 PDF 收藏，這個 extension 做得挺好。
